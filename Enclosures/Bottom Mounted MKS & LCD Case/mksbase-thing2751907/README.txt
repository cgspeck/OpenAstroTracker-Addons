# Summary

A box designed to contain the motherboard and a MosFet switch (Either of the two common ones found on eBay) for a 3D printer. Cable anchor positions are built in. The fan is a 40mm x 10mm one.
Added a modified lid and an enclosure to accommodate a 3.2 LCD touch screen for use with the MKS Gen L board. (https://www.aliexpress.com/snapshot/0.html?spm=a2g0s.9042647.6.2.1PZkP4&orderId=88640479325061&productId=32783143532)
